# Housing Market Intelligence Dashboard

A full-stack housing market analysis platform that bridges the gap between macroeconomic data and real-time social sentiment. This architecture leverages a Plotly Dash web interface for professional presentation and a heavy-duty GCP data engine for complex analytics and AI insights.

## The Vision: Value vs. Vibe
The platform identifies market inflections by correlating "Hard Data" (Zillow, FRED, Google Trends) with "Social Heat" (Reddit). It provides high-utility views for homebuyers and investors by revealing when the national conversation in real estate communities begins to lead or lag behind official economic reports.

## 1. Technical Architecture (GCP + Dash)
The platform utilizes a decoupled architecture to ensure both a polished user interface and robust automation.

| Component | Technology | Role |
|---|---|---|
| **Data Engine** | Google BigQuery | Centralized data warehouse for structured metrics & semi-structured social posts. |
| **Automation** | Cloud Scheduler | Triggers daily updates to fetch new Google Trends and Reddit data. |
| **Backend API** | Cloud Run / Flask | Hosts the Dashboard and exposes the `/api/run-pipeline` ETL trigger endpoint. |
| **Intelligence** | Vertex AI (Gemini) | LLM-driven engine generating Weekly Market Briefings and answering Q&A. |
| **Frontend UI** | Plotly Dash | Interactive, data-heavy dashboard with real-time AI integration. |

## 2. Data Sources & AI Integration

### Market Fundamentals (The Value)
*   **Zillow Research**: Monthly ZHVI (Home Value Index), Inventory, and Price metrics across US MSAs.
*   **FRED API**: Tracks 30-Year Fixed Rate Mortgage Averages and CPI to provide immediate financial context.

### Social Intelligence (The Vibe)
*   **Google Trends**: Weekly search interest scores for distress signals (e.g., "Foreclosure Auctions", "Home Insurance"). This data powers the custom **Composite Stress Index (CSI)**.
*   **Reddit Discourse**: Daily extraction of posts and top comments from 6 major real estate subreddits (e.g., `r/FirstTimeHomeBuyer`, `r/realestateinvesting`). 

### AI Intelligence Layer
*   **Vertex AI Briefings**: On-demand, audience-specific market briefings (Homebuyer, RE Investor, REIT) generated by Gemini using aggregated BigQuery context. Briefings are cached daily to minimize LLM costs.
*   **Interactive Q&A**: A dynamic chat interface allowing users to ask follow-up questions about the market data, with built-in daily rate limiting.

## 3. Dashboard Features
The dashboard features two high-impact modules:

1.  **Tab 1: Market Signals (Intelligence Panel)** 
    *   Features the AI Weekly Briefing, dynamic Q&A, and the Composite Stress Index gauge.
    *   Visualizes a 5-year trendline of forward-looking distress signals.
2.  **Tab 2: Fundamentals (Data Explorer)**
    *   Interactive US Map visualizing housing metrics by State and City.
    *   Deep-dive time series charts for specific MSAs to compare local growth.
    *   Economic indicator charts tracking Mortgage Rates and Inflation.

## 4. Implementation & Deployment

### GCP Setup
*   Enable BigQuery, Vertex AI API, Cloud Run, and Cloud Scheduler.
*   The Cloud Run service requires the `roles/aiplatform.user` IAM permission to call Gemini.
*   The Scheduler authenticates to the Cloud Run ETL endpoint using an `X-Pipeline-Secret` header.

### Environment Configuration
Required `.env` variables for local development:
*   `GCP_PROJECT_ID` & `GCP_DATASET_ID`
*   `GEMINI_MODEL` & `VERTEX_LOCATION`
*   `PIPELINE_SECRET`
*   `MAX_QA_PER_DAY`

### Running Locally
1. Start the Dash application:
```powershell
cd dashboard
pip install -r requirements.txt
python dash_app.py
```
2. Navigate to `http://127.0.0.1:8050`
